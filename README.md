# MobaMacOS Terminal

A [MobaXterm](https://mobaxterm.mobatek.net)-inspired SSH client for macOS, built with Java 17 + Swing.

![Alt text](./screenshot.png?raw=true "MobaMacOSTerm")

**Single-window workspace** — tabbed SSH sessions in the centre, a server tree on the right, and a file-transfer browser on the left. No Java required on the end-user's machine; the distributed `.app` / `.exe` bundles its own JRE.

---

## Features

| Area | Details |
|---|---|
| **SSH sessions** | Password, public-key (Ed25519, ECDSA, RSA), and ProxyCommand auth via SSHJ |
| **Local terminal** | Full PTY shell (`$SHELL --login`) via pty4j |
| **Server browser** | Folders, drag-and-drop reordering, `~/.ssh/config` auto-import |
| **File transfer** | Split-pane SFTP browser — upload/download with per-file progress bar |
| **CWD tracking** | Remote panel follows the terminal's working directory in real time (OSC 7) |
| **SSH tunnels** | Local port-forwarding via Tools → SSH Tunnels… |
| **Themes** | FlatLaf: IntelliJ (default), Darcula, FlatLight, FlatDark |
| **Session restore** | Remembers open sessions across restarts |

## Requirements

### To build
- JDK 17+ (JDK 14+ required for `jpackage` packaging targets)
- Maven 3.8+
- `make`

### To run the distributed app
- **Nothing.** The `.app` / `.exe` bundles a private JRE.

---

## Building

```bash
# Download dependencies
make deps

# Compile only
make compile

# Run tests
make test

# Build the fat JAR (all deps + resources bundled)
make jar

# Run directly (no packaging needed)
make run
```

---

## Distribution

### macOS — self-contained `.app`
```bash
make app-mac        # → dist/mac/MobaMacOS Terminal.app
```

### macOS — drag-to-install `.dmg`
```bash
make dmg            # → dist/mac/MobaMacOS Terminal-1.0.0.dmg
```

### Windows — installer `.exe`
Must be run on Windows with [WiX Toolset 3.x](https://wixtoolset.org/releases/) installed.
```bash
make app-win        # → dist/win/MobaMacOS Terminal-1.0.0.exe
```

### Show all targets
```bash
make help
```

> **Note:** `make app` auto-detects the current OS and delegates to `app-mac` or `app-win`.

| [Jackson](https://github.com/FasterXML/jackson) | 2.16.1 | JSON config persistence |

---

## Configuration

Settings are stored in `~/.mobamacos/config.json` and are created automatically on first run.
Servers imported from `~/.ssh/config` appear in a read-only **SSH Config** folder.
